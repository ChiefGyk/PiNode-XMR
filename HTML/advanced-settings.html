Mining

Note:

Mining, by design is CPU intensive and not suited to single board computers. Although not at all recommended on this device, it is possible.
This feature has been added for experimental/educational purposes and thought should be given for your devices CPU cooling before activating to prevent physical damage. This feature may also handicap the nodes speed verifying transactions/blocks.
*Note: Mining currently over clearnet only.

Update: Manual trigger for Monero update check

New Monero versions need to be compiled for the Raspberry Pi by the Monero dev team. There is usually a short delay between the main rollout of a new version and the Raspberry Pi compatible version. Due to the collaboratiorive nature of Monero the "--strip" option may need to be used. Consult reddit.com/r/PiNode for instructions.
