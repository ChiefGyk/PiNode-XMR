Update: Manual trigger for Monero update check

New Monero versions need to be compiled for the Raspberry Pi by the Monero dev team. There is usually a short delay between the main rollout of a new version and the Raspberry Pi compatible version. You node will auto-attempt update every Sunday 00:00, or use this to update sooner if available.
